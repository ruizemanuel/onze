# Onze

No-loss on-chain fantasy football on Celo — **World Cup 2026 edition**. Build your starting XI (11 players + captain) for each phase, compete for the prize pool funded by Aave V3 yield on a USDT seed, and lose nothing: your $1 deposit is always withdrawable. Picks from an ERC-8004 verified AI Coach.

Live: https://onze-beta.vercel.app

## Stack
Hardhat + Next.js 16 + Tailwind v4 + wagmi/viem + Aave V3 (Celo) + ERC-8004 (agent trust) + Neon Postgres

## Develop
```bash
pnpm install
pnpm dev
```
